- `forbidden_actions.md` — forbidden actions and disallowed behaviors (FA-*).
- `malicious_adversarial_use.md` — malicious/adversarial use guardrails (GA-MA-*).
- `sensitive_restricted_domains.md` — sensitive/restricted domain guardrails (GA-SD-*).
- `hallucination_inference_boundaries.md` — hallucination/inference boundaries (GA-HI-*).
- `escalation_hitl.md` — escalation, refusal behavior, and interaction termination rules (GA-ER-*).
- `guardrail_providers.md` — GraniteGuardianTool categories + RiskAgent risk-id selection (taxonomy mapping).
- `enforcement_matrix.md` — enforcement matrix (provider signal → action).
- `never_do_rules.md` — non-negotiable “never do” list.
- `open_questions_residual_risks.md` — open questions and residual risks.
- `referenced_standards.md` — informative norms/standards referenced.
