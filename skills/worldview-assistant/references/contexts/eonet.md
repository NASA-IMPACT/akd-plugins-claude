# EONET (Event context)

## Name
EONET

## Purpose
Provides event context that can be attached to visualization/exploration (e.g., wildfire, storms, volcanoes).

## Data provided (as described)
- Event title
- Geometry
- Time window
- Category
- External source links

## Permissions / auth model
Public-only.
